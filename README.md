## 🕹 How to Use

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/jlebre/42So_Long.git
cd 42So_Long
```

### 2️⃣ Compile the Game
```bash
make
```

### 3️⃣ Run the Game
```bash
./so_long maps/sample_map.ber
```
Replace `sample_map.ber` with your own map file.

### 4️⃣ Controls
- `W` - Move up
- `A` - Move left
- `S` - Move down
- `D` - Move right
- `ESC` - Quit the game
